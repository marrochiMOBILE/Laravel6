# Controller resource
adalah keperluan untuk pembuatan crud

cara pembutanya :
```
php artisan make:controller NamaControllernya -r
php artisan make:controller NamaControllernya --resource
```
sama sajja mau pilih yg atas atau bawah.

cara pemanggilanya di routes:

```php
Route::resource('path', 'controllernya');
```
contoh buat yg ditas
```php
Route::resource('supplier', 'Csupplier');
```
